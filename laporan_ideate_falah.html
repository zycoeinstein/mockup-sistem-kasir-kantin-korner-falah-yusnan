1. Ringkasan Problem Statement & HMW

Proses transaksi dan pencatatan stok di Kantin Korner masih dilakukan manual, menyebabkan kesalahan perhitungan dan keterlambatan laporan. Dibutuhkan aplikasi kasir berbasis web agar pencatatan transaksi, stok, dan laporan dapat berjalan lebih efisien dan akurat.

How Might We (HMW)

  • Bagaimana membantu kasir mencatat transaksi cepat dan akurat?
  • Bagaimana mempermudah pengelola memantau stok secara real-time?
  • Bagaimana menyediakan laporan penjualan otomatis?
  • Bagaimana mengurangi kesalahan pencatatan transaksi harian?

2. Brainstorming Ide

No Ide Solusi Deskripsi Singkat
1Sistem kasir onlinePencatatan transaksi berbasis web untuk kasir.
2Pendataan stok otomatisUpdate stok secara otomatis setiap transaksi.
3Laporan keuangan digitalGenerate laporan otomatis harian dan bulanan.

3. Sketsa Alur Sistem

Kasir login → Input transaksi → Sistem cek stok → Jika stok habis → Peringatan & tolak transaksi → Jika stok tersedia → Simpan transaksi → Update stok otomatis → Laporan diperbarui → Data bisa diakses pengelola.

4. Refleksi

Tantangan: Menentukan fitur prioritas yang penting tanpa membuat sistem terlalu kompleks.

Alasan dipilih: Solusi ini paling sesuai untuk meningkatkan efisiensi operasional kantin kampus.

Relevansi: Menerapkan digitalisasi bisnis dan manajemen data untuk lingkungan pendidikan.
